
# GitHub Issue Reporting Guide

## How to Report Issues

### Creating a New Issue

1. **Navigate to the Repository**
    - Go to the GitHub repository for WaveForge Pro

2. **Click the Issues Tab**
    - Select "Issues" from the repository navigation

3. **Click "New Issue"**
    - Green button in the top right

4. **Fill in the Details**
    - **Title**: Brief description of the problem
    - **Description**: Detailed explanation including:
      - Steps to reproduce
      - Expected behavior
      - Actual behavior
      - Screenshots (if applicable)
      - System information (OS, version)

5. **Add Labels** (optional)
    - Select relevant tags like `bug`, `enhancement`, or `documentation`

6. **Assign Priority**
    - Set severity if your repository uses it

7. **Click "Submit New Issue"**

### Best Practices

- Search existing issues before creating a new one
- Provide clear, reproducible steps
- Include error messages and logs
- Be specific about your environment
